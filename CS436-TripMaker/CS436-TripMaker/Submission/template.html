Trip Maker
External link to .apk
External link to Zip File
Trip Maker allows the user to identify an area to visit and what they are interested in visting in the area. The app then generates a travel itinerary. The draft itinerary includes 10 hours of activities per day, but the user can edit that to add or take away activities. The itinerary includes estimates on how long the user should spend doing each activity. The user then confirms the list of activities to create the final itinerary. Itineraries are saved to the user's account, and the user has the chance to add photos from the trip along with the itinerary.
